Selitys koodista ja ohjeet

Generoi numeroita käyttäjän antamasta kuvasta, jossa on numerot 0-9.

Ohje:
Koodissa on pakko olla
Dim numbers(9) ennen LoadNumbers funktiota
 
LoadNumbers([file$])
-Kuvatiedosto, jossa numerot on (Valinnainen). Oletus on num.png
 
DrawNumbers(numero, x, y, length)
- Arvo, jonka haluat tulostaa.
- X ja Y, arvon sijainti näytöllä
- length, Numeroiden lukumäärä
 
Kutsu ennen DrawNumbers funktiota LoadNumbers(kuvatiedoston polku). Sitten
voit kutsua funktiota esim. DrawNumbers(80, 20, 10, 5). Alla on esimerkki koodissa.
 
Tarvitsee kuvan www.jokejewel.org/num.png

'---------Funktiot------------------------------
'Ohje:
'Koodissa on pakko olla
'Dim numbers(9) ennen LoadNumbers funktiota

'LoadNumbers([file$])
'-Kuvatiedosto, jossa numerot on (Valinnainen). Oletus on num.png

'DrawNumbers(numero, x, y, length)
'- Arvo, jonka haluat tulostaa.
'- X ja Y, arvon sijainti näytöllä
'- length, Numeroiden lukumäärä
'-----------------------------------------------

Function LoadNumbers(file$="C:\num.png")
    num=LoadImage(file$)
    numw=ImageWidth(num)/10
    numh=ImageHeight(num)

    For i=0 To 9
        numbers(i)=MakeImage(numw, numh)
        DrawToImage numbers(i)
            id=i
            DrawImageBox num,0,0,id*numw,0,numw, numh
        DrawToScreen
    Next i

    DeleteImage num
End Function

Function DrawNumbers(number, x, y, length)
    num$=String("0",length-Len(Str(number)))+Str(number)
    numw=ImageWidth(numbers(0))
    numh=ImageHeight(numbers(0))

    pituus=Len(num$)
    If length<pituus Then length=pituus

    For i=0 To length-1
        DrawImage numbers(Int(Mid(num$, i+1, 1))), x+i*numw, y, 0, 0
    Next i

    Color 0,0,0
    Box x, y, length*numw, numh, OFF
End Function
