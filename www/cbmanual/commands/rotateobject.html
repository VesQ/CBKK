Objektit : komennot

ROTATEOBJECT


   KUVAUS

Kääntää objektin täsmälleen haluttuun kulmaan. Välittömästi.

Kulma tulisi antaa väliltä 0-360.

   KÄYTTÖ
ROTATEOBJECT objektimuuttuja, kulma

  • objektimuuttuja = Se muuttuja, mihin objekti ladattiin tai luotiin.
  • kulma = Kokonaisluku tai desimaaliluku. Mihin kulmaan objekti käännetään.

    • Katso myös: TURNOBJECT, POINTOBJECT, CLONEOBJECTORIENTATION

       ESIMERKKI
    FrameLimit 40 'limit game speed

    soldier=LoadObject("Media\soldier.bmp",36)

    'make another object
    soldier2=CloneObject(soldier)
    PositionObject soldier2,0, -100

    ClsColor cbwhite

    Repeat

        'update this code every 1.5 seconds
        If TIMER()>moment+1500 Then
            'increase soldier angle
            RotateObject soldier,ObjectAngle(soldier)+10
            
            'rotate to random angle
            RotateObject soldier2,Rand(360)
            
            moment=TIMER()
        EndIf

        DrawScreen

    Until EscapeKey()
