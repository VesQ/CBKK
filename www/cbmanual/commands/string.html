Rakentaa uuden merkkijonon toistamalla annettua merkkijonoa annettun määrä kertoja.

   KÄYTTÖ
STRING (jono, määrä)

  • jono = Toistettava merkki tai merkkijono.
  • määrä = Montako kertaa merkkijonoa toistetaan. Kokonaisluku.

    •    ESIMERKKI
    AddText "Use LEFT and RIGHT arrow to change length"

    length=1

    Repeat

        If KeyHit(cbkeyleft) Then length-1
        If KeyHit(cbkeyright) Then length+1
        
        If length<0 Then length=0

        build$=String("Hello",length)
        
        Text 0,50,build$
        
        DrawScreen
        
    Until KeyHit(cbkeyreturn)
